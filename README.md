# ☕ The Daily Drip

A multi-page coffee shop website built as a web development project, featuring a browsable menu, user account pages, and a clean, cozy visual identity.

## About

The Daily Drip is a front-end website for a fictional coffee shop, designed to give visitors an easy way to explore the menu, learn about the shop, sign in or create an account, and leave a rating. The project was built collaboratively by a team of two, with a strong focus on front-end design, layout, and styling.

## Pages

- **Home** – Landing page introducing The Daily Drip
- **Menu** – Split across three categories: Drinks, Desserts, and Breakfast
- **About Us** – Background on the coffee shop
- **Sign In / Sign Up** – User account pages
- **Rate Us** – Page for customers to leave feedback

## Tech Stack

- HTML5
- CSS3
- Custom imagery for menu items and page backgrounds

## My Role

I focused primarily on the **front-end** of the project — building out the page structure, styling, and overall visual experience across the site (menus, layout, backgrounds, and UI consistency), while collaborating with my teammate on the remaining functionality.

## Team

- 2 members
- Built as a collaborative web development project

## Getting Started

Since this is a static HTML/CSS site, you can run it locally with no setup:

1. Clone or download the repository
2. Open `homePage.html` in your browser

No build tools or dependencies required.
